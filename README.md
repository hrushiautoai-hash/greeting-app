# Greeting App 👋

A simple web application built with HTML, CSS and Node.js

## What Does This App Do?
- User types their name
- Clicks "Say Hello" button
- App replies with a greeting message

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
- Connection: REST API

## How to Run This Project

### Step 1 - Start Backend:
cd backend
node server.js

### Step 2 - Open Frontend:
Open frontend/index.html with Live Server
